# 🚀 SkillSync Project Engine

SkillSync Project Engine is an AI-powered machine learning application that recommends suitable technical project domains based on student interests, skills, favorite subjects, experience level, and career goals.

## Features
- Machine Learning based recommendation system
- Dynamic project domain prediction
- Suggested project ideas
- Recommended technology stack
- Interactive Streamlit web interface

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Streamlit
- Random Forest Classifier

## Supported Domains
- Machine Learning
- Web Development
- Cybersecurity
- Cloud Computing
- Java Development
- Data Analytics
- Android Development

## How to Run

```bash
streamlit run app.py
